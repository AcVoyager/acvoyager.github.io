What is CDN?

Please see the Microsoft official document.

A content delivery network (CDN) is a distributed network of servers that can efficiently deliver web content to users. CDNs’ store cached content on edge servers in point-of-presence (POP) locations that are close to end users, to minimize latency.

BGP

Azure CDN can also accelerate dynamic content, which cannot be cached, by leveraging various network optimizations using CDN POPs. For example, route optimization to bypass Border Gateway Protocol (BGP).

So what is BGP?

BGP is on Application Layer.

Border Gateway Protocol (BGP) is a standardized exterior gateway protocol designed to exchange routing and reachability information among autonomous systems (AS) on the Internet.[1] BGP is classified as a path-vector routing protocol,[2] and it makes routing decisions based on paths, network policies, or rule-sets configured by a network administrator.

See Border Gateway Protocol

Steps of Azure CDN Network

The detailed steps can be found in the document. Here putting some important steps:

  • It the DNS that routes the request to the best performing POP location, which is usually the POP that is geographically closest to the user.
  • The user interacts with point-of-presence(POP), and POP interacts with origin server.
  • Whether POP interacts with origin server depends on the time-to-live(TTL) of the requested files. The default TTL is 7 days.
    • My inference: When the file get updated at the origin server, the user may not be able to access the latest version if the file in the POP has not expired TTL.

Java和C++的区别是什么样的？

Java和python的区别 Java 和C++区别

个人认为最容易想到的点：

  • Java和Python，很明显，前者静态语言，后者动态语言。Java首先被编译为字节码，然后解释运行；Python每次运行时都需要解释。
  • Java和C++，前者编译为字节码，以.class的形式存储在disk里，需要时通过类加载在JVM上运行；后者直接编译、链接为二进制代码运行。

静态/动态语言，强弱类型

静态类型 vs 动态类型：区别在于type checking何时发生

  • 静态类型语言：在程序运行之前，代码中的变量会被检查（编译器），错误会被查出。例如C++，Java
  • 动态类型语言：在程序运行之前是不会做检查的，而是在运行时检查。如果代码有错，会在程序运行时报错。动态语言中，变量没有类型，值有类型。 例如Python.
  • 参考：Difference between static and dynamic programming languages
